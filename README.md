Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install lib_saprfc


Or clone the repo:

    $ git clone https://erasmo_montes@bitbucket.org/erasmo_montes/lib-saprfc.git 
    $ python setup.py install
