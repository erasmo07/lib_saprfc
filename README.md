Lib Sap RFC
===============================

version number: 0.0.1
author: Erasmo Montes

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install lib_saprfc


Or clone the repo:

    $ git clone https://github.com/emontes/lib_saprfc.git
    $ python setup.py install
